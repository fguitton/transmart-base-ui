To load studies, connect to a data source using the "Add Data Source" form in the sidebar.
  1. Select a data source from the defaults or enter its URL address and choose a name for it.
  2. Request an authorization code. You will be redirected to a login page for the chosen data source. Once the code is obtained, paste in the designated area.
  3. Click save.
  1. To browse studies, click on the folder icon next to one.
  2. Concepts can then be dragged into the cohort selection tab.
